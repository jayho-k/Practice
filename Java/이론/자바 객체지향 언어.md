## 2. 클래스와 객체

- 객체를 정의해 놓은 것 ==> 용도 : 객체를 생성하는데 사용

- 클래스 : 제품설계도 , 객체 : 제품 ==> 설계도 한번 만들어 놓으면 제품 쉽게 만들수 있어서



#### 속성과 기능

- 객체 : 속성(변수) + 기능(method)



#### 객체와 인스턴스 (거의 같은 말)

인스턴스 : 특정 클래스로부터 생성된 객체
